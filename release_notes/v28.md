# Xion v28 Release Notes

## Overview

The Xion v28 release introduces ZK email and DKIM modules, codec improvements, and various bug fixes. The recommended version is v28.1.0.

## What's Changed

- Added codecs for DKIM and ZK modules
- Fixed ZK email public inputs extraction and field ranges
- Fixed upgrade handler issues
- Hardened release type detection and downstream triggers
- Upgraded cosmos-sdk to v0.53.6
- Upgraded cometbft to v0.38.21
- Upgraded ibc-go to v10.5.0
- Upgraded CosmWasm/wasmd to v0.61.8

## Upgrade Information

- **Recommended Version**: v28.1.0
- **Chain ID**: xion-testnet-2
