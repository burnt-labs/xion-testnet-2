# Xion v22.0.0 Release Notes

## Overview

Xion v22.0.0 is a critical security and stability release that addresses several important vulnerabilities and adds key improvements to the platform. Released on November 6, 2025.

## What's Changed

### Security & Bug Fixes

- **Integer Overflow Fix**: Resolved an integer overflow vulnerability in platform fee calculations that could have led to incorrect fee handling
- **WASM Panic Fix**: Fixed a WASM panic issue affecting module functionality and stability
- **JWK Validation**: Addressed JWK (JSON Web Key) validation problems to improve authentication security
- **Minimum Fee Handling**: Corrected minimum fee handling to ensure proper transaction fee enforcement
- **WebAuthN Size Limit**: Implemented a size limit for WebAuthN data to prevent potential overflow attacks and improve security

### CLI Enhancements

- **Delete Audience Claims Command**: Implemented the previously missing CLI command for deleting audience claims, providing operators with better control over claim management

### Infrastructure Updates

- **Go Module Dependencies**: Updated Go module dependencies to latest stable versions
- **Global Fee Ante Handler**: Fixed the global fee ante handler with comprehensive tests to ensure correct fee processing
- **IBC Hop Limitation**: Limited IBC (Inter-Blockchain Communication) hops to a maximum of 10 to prevent potential abuse
- **TypeScript Package**: Removed TypeScript package publishing from the repository for better maintenance focus

### Documentation

- **WebAuth Documentation**: Enhanced documentation detailing how WebAuth is utilized within the Xion module, improving developer experience
