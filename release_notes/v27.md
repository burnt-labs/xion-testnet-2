## Upgrade Instructions

### For Validators

1. **Download the Binary**
   ```bash
   # Download for your platform
   wget https://github.com/burnt-labs/xion/releases/download/v27.0.0-rc2/xiond_27.0.0-rc2_linux_amd64.tar.gz
   
   # Verify checksum
   sha256sum xiond_27.0.0-rc2_linux_amd64.tar.gz
   
   # Extract and install
   tar -xzf xiond_27.0.0-rc2_linux_amd64.tar.gz
   sudo mv xiond /usr/local/bin/
   ```

2. **Verify Installation**
   ```bash
   xiond version
   # Should output: v27.0.0-rc2
   ```

3. **Upgrade Process**
   - The upgrade will be coordinated via governance proposal
   - Node will halt at the specified upgrade height
   - Replace the binary and restart the node
   - The upgrade handler will run automatically

### For Node Operators

The upgrade includes:
- Automatic state migration
- Module initialization updates
- No manual intervention required (beyond binary replacement)
