# Xion v25 Release Notes

## Overview

The Xion v25 release introduces improvements to indexer testing, CI/CD automation, and workflow triggers. This release focuses on enhancing the testing infrastructure and automating asset updates. The recommended version is v25.0.0.

## What's Changed

### v25.0.0

#### Indexer Improvements

- **Fixed Indexer Tests**: Resolved issues with indexer test execution and validation
- **Indexer Configuration Checks**: Added proper configuration validation for enabling indexer functionality
- **Test Structure Improvements**: Restructured new tests using existing patterns for better consistency

#### CI/CD and Automation Enhancements

- **Automated Asset Updates**: Added workflow trigger to automatically update xion-assets repository with new release versions
- **Docker Build Trigger Fix**: Corrected issues with Docker build workflow triggers to prevent unwanted executions
- **Workflow Optimization**: Refined trigger conditions to ensure workflows fire only when desired

#### Release Management

- **Release Branch**: Merged v24 release branch and prepared v25 release structure
- **Version Bump**: Updated version to v25.0.0
